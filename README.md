# Weather Bot

This is a Weather bot. It is written in python and uses the python-telegram-bot library and [weather api](https://openweathermap.org/api).

## Menu map

### Cities

- Samarkand
- Bukhara
- Jizzakh
- Tashkent

### Send Weather info

Example:

- Kun: Dushanba: Ochiq osmon
- Ertalab: 6°C ☀
- Kun bo`yi: max-min: 14°C - 6°C ☀
- O`rtacha: 14°C ☀
- Kechqurun: 9°C 
- Shamol: 2 m/s
